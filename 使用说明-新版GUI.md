## 🎯 完整使用流程

### 步骤1：启动程序
```bash
python link_monitor_modern_gui.py
```

### 步骤2：配置文件
1. **选择CSV文件**：点击"浏览"，选择包含链接的CSV文件
2. **选择输出目录**：点击"浏览"，选择报告保存位置

### 步骤3：可选配置
- 如需定时运行：勾选"启用定时运行"并设置时间

### 步骤4：开始监测
1. 点击 **"▶ 立即开始监测"**
2. 等待浏览器自动打开
3. **重要**：浏览器会先打开小红书主页并等待10秒
   - 如已登录：无需操作
   - 如需登录：在10秒内完成扫码登录

### 步骤5：自动执行
程序将自动：
- ✅ 访问CSV中的每个链接
- ✅ 等待页面加载（3秒）
- ✅ 截图并保存
- ✅ 生成Excel报告
- ✅ 关闭浏览器

### 步骤6：查看结果
- 完成后会弹出提示框
- 查看输出目录中的Excel报告
- 报告包含：链接、状态、截图、时间

1. 勾选"启用定时运行"
2. 设置运行时间（如：09:00）

## 🔧 故障排除

### 问题1：浏览器未安装
```bash
python -m playwright install chromium
```

### 问题2：CSV读取失败
- 确保CSV文件编码为UTF-8
- 第一行为标题"链接"
- 每行一个链接

### 问题3：截图保存失败
- 检查输出目录是否有写入权限
- 确保磁盘空间充足

### 问题4：Excel生成失败
- 确保已安装openpyxl
- 检查输出目录权限

## 💡 使用技巧

1. **首次使用**：先用2-3个链接测试
2. **登录保持**：登录一次后会保留状态
3. **批量处理**：建议每次不超过50个链接
4. **定期清理**：定期清理output目录
5. **备份报告**：重要报告及时备份
