## 📖 使用教程

### 1. 启动程序
```bash
python link_monitor_pro_gui.py
```

### 2. 选择模式
- **快速模式**：适合大量链接快速扫描
- **完整模式**：适合详细分析（推荐）

### 3. 配置参数
- CSV文件：包含小红书链接
- 输出目录：保存结果位置
- 扫码时间：建议45-60秒（Pro模式需要更多时间）

### 4. 开始监测
点击"立即开始监测"，程序会：
1. 打开浏览器
2. 等待扫码登录
3. 逐个访问链接
4. 抓取详细内容
5. 下载封面图片
6. 生成Excel报告

## 🔧 技术实现

### 封面图抓取
```python
# 多选择器策略
selectors = [
    'img[src*="sns-img"]',      # 小红书图床
    '.carousel img',            # 轮播图
    '[class*="swiper"] img',    # 滑动图
    '.note-content img'         # 笔记内容图
]

# 去重下载
for img in images:
    if 'sns-img' in src:        # 仅下载小红书图片
        img.screenshot()        # 截取图片元素
```

### 悬停交互
```python
# 查找作者元素
author = page.query_selector('.author-container')

# 悬停触发
author.hover()

# 等待加载
time.sleep(1)
```

## 🐛 注意事项

### 1. 扫码时间
- Pro模式建议设置 **45-60秒**
- 需要等待页面完全加载
- 等待动态内容渲染

### 2. 处理速度
- Pro模式比基础版慢1倍
- 每个链接需要10-15秒
- 建议小批量处理（10-20个链接）

### 3. 封面图抓取
- 依赖页面DOM结构
- 小红书更新可能影响抓取
- 最多抓取10张（防止过多）

### 4. 互动数据
- 数据为抓取时的实时数据
- 可能因页面加载不完整而缺失
- 悬停操作可能不总是成功

## 📞 问题排查

### Q: 封面图下载失败
**A**: 检查网络连接，确保页面完全加载

### Q: 互动数据为空
**A**: 可能页面未加载完成，增加等待时间

### Q: 悬停无效
**A**: 小红书页面结构可能变化，检查选择器

### Q: Excel很大
**A**: 封面图较多导致，可选择快速模式
