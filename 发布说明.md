## 🚀 快速开始

### 1. 首次使用（必须！）

在运行主程序之前，**必须先安装浏览器**：

```
双击运行：安装浏览器.bat
等待2-5分钟完成安装
```

### 2. 运行程序

```
双击运行：WPP_MD_链接监测工具.exe
```

### 3. 界面操作

1. **选择CSV文件** - 包含要监测的小红书链接
2. **选择输出目录** - 报告和截图保存位置
3. **设置扫码时间** - 建议30-60秒
4. **点击开始监测** - 自动完成所有操作

## 🔧 常见问题

### Q: 提示"浏览器未安装"
**A**: 运行 `安装浏览器.bat`

### Q: 程序打不开
**A**: 检查Windows Defender是否阻止，添加信任

### Q: 截图是黑屏
**A**: 增加扫码等待时间到45-60秒

### Q: CSV读取错误
**A**: 确保UTF-8编码，第一行是"链接"
