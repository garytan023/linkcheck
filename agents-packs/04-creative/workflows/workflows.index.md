# 04-creative Workflows

## Workflow 1: 竞品爆款拆解

### 输入
- 竞品账号/链接
- 目标人群

### 步骤
1. 抓取爆款内容
2. 拆解结构（钩子/内容/CTA）
3. 提取可复用公式
4. 输出优化建议

### 输出
- 爆款元素清单
- 公式化模板
- 落地建议

---

## Workflow 2: 脚本批量生成

### 输入
- 产品信息
- 目标人群
- 渠道平台
- 数量要求

### 步骤
1. 人群细分
2. 痛点挖掘
3. 脚本撰写（多版本）
4. 分镜/拍摄建议

### 输出
- 脚本表（标题/前3秒/镜头/旁白/字幕/CTA）
- 视觉参考关键词
