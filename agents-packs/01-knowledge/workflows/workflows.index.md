# 01-knowledge Workflows

## Workflow 1: 主题研究（30分钟输出一页纸）

### 输入
- 研究主题
- 时间范围
- 重点关注点

### 步骤
1. 搜索相关资料（多源）
2. 筛选可信来源
3. 提炼关键信息
4. 结构化输出

### 输出
- TL;DR（3-5条）
- 关键结论
- 证据点（带来源）
- 待确认项
- 下一步建议

---

## Workflow 2: 资料入库

### 输入
- 文章/文档/会议纪要

### 步骤
1. 提炼核心观点
2. 提取关键数据
3. 打标签分类
4. 存入知识库

### 输出
- 摘要
- 标签列表
- 原文链接
- 相关主题关联
