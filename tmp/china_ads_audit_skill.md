# 竞价广告全平台审计 Skill

> 适用于 Claude Code / OpenClaw

## 概述

本 Skill 提供**全平台竞价广告一键审计**能力，覆盖中国主流投放平台，帮助快速诊断广告投放健康度。

## 支持平台

| 平台 | 广告产品 | 核心指标 |
|------|---------|---------|
| 抖音 | 千川、信息流、星图 | ROI/ROAS、转化成本、互动率 |
| 小红书 | 聚光广告、薯条 | 阅读成本、互动率、转化成本 |
| 腾讯 | 朋友圈、视频号、腾讯广告 | CTR、CPM、转化成本 |
| 京东 | 京准通、京东联盟 | ROI、ROAS、ACOS |
| 天猫 | 万相台、超级互动城 | ROI、互动率、收藏加购率 |
| 百度 | 搜索、信息流、爱采购 | CTR、转化成本、ROI |
| 快手 |磁力引擎、信息流 | ROI、转化成本、互动率 |
| 微信 | 微信广告、视频号 | CTR、CPM、转化成本 |

## 核心指标定义

### 通用指标

| 指标 | 英文 | 计算方式 | 优秀值 | 预警值 |
|------|------|---------|--------|--------|
| 投入产出比 | ROI | 销售额/广告费 | >3 | <1.5 |
| 广告投入产出比 | ROAS | 广告收入/广告费 | >3 | <1.5 |
| 千次展示成本 | CPM | 广告费/展示次数×1000 | <50 | >100 |
| 单次点击成本 | CPC | 广告费/点击次数 | <2 | >5 |
| 单次转化成本 | CPA/转化成本 | 广告费/转化数 | <50 | >150 |
| 点击率 | CTR | 点击次数/展示次数 | >1.5% | <0.5% |
| 转化率 | CVR | 转化数/点击次数 | >2% | <0.5% |

### 平台特色指标

| 平台 | 特色指标 | 优秀值 |
|------|---------|--------|
| 抖音 | 5秒完播率、互动率、GPM | >20%、>5%、>50 |
| 小红书 | 笔记阅读量、收藏率、分享率 | 因品类而异 |
| 京东 | ACOS、ROAS、搜索排名 | ACOS<30% |
| 天猫 | 收藏加购率、转化率 | >10%、>3% |
| 百度 | 搜索占有率、对话成本 | 因行业而异 |

## 审计检查项

### 1. 账户基础设置（20%权重）

- [ ] 投放目标与业务目标一致
- [ ] 预算设置合理（不超投、不空耗）
- [ ] 出价策略匹配投放目标
- [ ] 投放时间设置正确
- [ ] 地域定向合理
- [ ] 设备定向优化
- [ ] 转化追踪配置正确
- [ ] pixel/SDK 安装验证

### 2. 定向策略（20%权重）

- [ ] 人群定向不過窄/過寬
- [ ] 兴趣定向与产品匹配
- [ ] 行为定向数据新鲜度
- [ ] 再营销受众配置
- [ ] 相似受众扩展合理
- [ ] 排除人群设置完整

### 3. 创意素材（25%权重）

- [ ] 素材规格符合平台要求
- [ ] 创意吸引力足够
- [ ] CTA 清晰明确
- [ ] 落地页加载速度
- [ ] 落地页与广告一致
- [ ] A/B 测试配置
- [ ] 素材更新频率

### 4. 投放策略（20%权重）

- [ ] 出价竞争力充足
- [ ] 预算分配合理
- [ ] 多广告组分散风险
- [ ] 测试新素材比例
- [ ] 季节/大促调整
- [ ] 竞品对抗策略

### 5. 数据分析（15%权重）

- [ ] 关键指标监控频率
- [ ] 异常预警机制
- [ ] 归因窗口合理
- [ ] 跨渠道归因
- [ ] ROI/ROAS 达标率
- [ ] 优化动作记录

## 各平台详细检查项

### 小红书聚光广告

```
检查维度：
├── 投放形式：信息流 / 搜索 / 话题
├── 核心指标：阅读成本、互动率、转化成本
├── 薯条投放：单篇笔记投放效果
├── 内容质量：笔记互动率、收藏率、分享率
└── 投放人群：精准度、竞品人群覆盖
```

### 抖音千川

```
检查维度：
├── 投放目标：短视频带货 / 直播带货 / 门店引流
├── 核心指标：ROI、ROAS、GPM、转化成本
├── 素材分析：完播率、互动率、5秒留存
├── 达人合作：挂车数据、ROI
└── 投放策略：莱卡定向、DMP人群
```

### 腾讯广告

```
检查维度：
├── 投放位置：朋友圈、视频号、公众号、小程序
├── 核心指标：CPM、CTR、CPC、转化成本
├── 定向能力：兴趣标签、行为数据、LBS
├── 创意形式：卡片、视频、轮播
└── 转化路径：蹊径、落地页、小程序
```

### 京东京准通

```
检查维度：
├── 投放产品：搜索推广、触点推荐、品牌聚效
├── 核心指标：ROI、ACOS、ROAS、搜索排名
├── 关键词：出价、匹配、质量分
├── 商品投放：SKU效果、竞品拦截
└── 大促配合：京喜、618、双11
```

### 天猫万相台

```
检查维度：
├── 投放场景：消费者运营、货品运营、场景运营
├── 核心指标：ROI、收藏加购率、转化率
├── 投放策略：人群智选、货品加速
├── 素材要求：主图视频、sku图
└── 数据分析：生意参谋关联
```

### 百度搜索

```
检查维度：
├── 投放类型：搜索、信息流、品牌专区
├── 核心指标：CTR、对话成本、ROI
├── 关键词：质量度、匹配模式、出价
├── 创意：闪投、组件、高级样式
└── 落地页：基木鱼、爱采购店铺
```

## 健康评分算法

```
总得分 = 账户基础×20% + 定向策略×20% + 创意素材×25% + 投放策略×20% + 数据分析×15%

评分等级：
A (90-100): 优秀，仅需微调
B (75-89): 良好，存在优化空间
C (60-74): 一般，需要关注
D (40-59): 较差，需重点优化
F (<40): 危险，需立即调整
```

## 注意事项

⚠️ **数据安全**
- 所有账号信息本地存储，不上传云端
- 定期更换 API Token
- 敏感数据脱敏处理

⚠️ **平台政策**
- 遵守各平台广告政策
- 规避违禁词、限流词
- 合规使用人群定向

⚠️ **数据延迟**
- 各平台数据延迟不同
- 抖音：T+1
- 小红书：T+1
- 腾讯：实时/ T+1
- 京东：T+1
