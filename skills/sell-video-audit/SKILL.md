# Sell Video Audit（详细版）

用**成交导向**而不是审美导向审查内容。先判断内容是否具备卖货能力，再指出最影响成交的结构问题，给出可执行修改建议或竞品策略洞察。

---

## 模式一：快审模式（单条视频）

### Step 1. 识别输入类型

判断用户给的是哪一种：
- 成片视频 / 视频链接
- 音频/转写稿
- 脚本/口播稿
- 分镜/拍摄计划

如果只有文字脚本没有画面，**明确说明"只能审查结构与表达，不能验证画面证据是否成立"**。

### Step 2. 判断内容目标

判断这条内容更接近哪种：
- **流量片**：目标是停留、播放、互动
- **种草片**：目标是建立好感和兴趣
- **成交片**：目标是下单或强转化
- **过渡型**：兼顾种草和转化，两头都不够强

用户没说明目标时，默认按"成交片"标准审查，在结论里注明这个假设。

### Step 3. 拆分三段结构

- **前段**：抢注意力（前 3 秒）
- **中段**：产品进入与使用展开
- **后段**：效果证明与促单

主类型判断：
- 痛点型 / 剧情/场景型 / 人设/身份型 / 效果验证型 / 价格促单型 / 混合型

### Step 4. 五个维度打分

总分 100 分，固定维度：

| 维度 | 分值 |
|------|------|
| 钩子强度 | /20 |
| 画面有效性 | /25 |
| 卖点可视化 | /20 |
| 转化闭环 | /20 |
| 测试潜力 | /15 |

评分规则：
- 先给分，再给 1-3 条证据
- 证据具体到镜头、表达、段落或动作
- 不说"不错""一般"这种空话

详细评分口径见 `references/rubric.md`。

### Step 5. 强制检查四个关键问题

每次必须回答：

1. **前 3 秒为什么值得继续看？**
2. **这条是在"说卖点"还是"证明卖点"？**
3. **用户看完后是否知道：为什么买、凭什么信、为什么现在买？**
4. **这条能否拆出多版本测试，还是只能赌单条？**

### Step 6. 输出格式

始终按此顺序输出：

1. 一句话结论
2. 总分与分项分
3. 当前更像哪种片
4. 主成交逻辑（主钩子/主卖点/主证据/主促单点）
5. 最影响成交的 3 个问题
6. 最值得保留的 3 个优点
7. 修改建议（必须改/建议改/可测试项）
8. 重拍/重剪镜头方案（如有必要）

**禁止**：
- 把"播放高"等同于"能卖货"
- 把"画面高级"等同于"成交强"
- 虚构视频中未出现的镜头或数据

## 模式二：详审模式（竞品批量分析）

当用户给一批视频（5–20条）做横向分析，或要求分析某品牌打法时，进入此模式。

### Step 1. 逐条快速归类

先对每条视频做快速判断（参考`references/rubric.md`的口径），填入汇总表：

| # | 品牌/视频 | 当前片型 | 成交潜力 | 核心卖点 | 最大问题 | 可抄模板 | 模仿指数 |
|---|---------|---------|---------|---------|---------|---------|---------|
| 1 | | | 高/中/低 | | | | /10 |

片型分类：流量片 / 种草片 / 成交片 / 过渡型 / 品牌活动片 / 促销引流片

### Step 2. 识别品牌打法（共性提炼）

从以下维度归纳每个品牌的打法特征：

**A. 单卖点成交逻辑**
- 一条只讲一个明确理由
- 用户收益直接
- 购买理由清楚
- 代表：华为续航片、防诈片

**B. 场景种草转成交逻辑**
- 不是说卖点，而是证明卖点
- 把功能放进真实羡慕场景
- 兼顾好看和说服力
- 代表：vivo 演唱会长焦、扫街长焦

**C. 高势能人设种草逻辑**
- 想要感强
- 容易建立品牌高级感
- 适合新品期拉高势能
- 代表：明星×产品绑定、专业人设开箱

### Step 3. 强制分析四个共性问题

1. **讲概念，不做证明**——如"电影感""专业V单""幸福感"，是否有对应证据画面？
2. **行动理由不够强**——是否回答了"为什么现在买"？
3. **促销片信任承接不足**——优惠/分期是否叠加了产品力和场景？
4. **预告感 vs 成交感**——结尾是引导预约/下单，还是只停在"发布会见"？

### Step 4. 提炼可套用模板

从样本中归纳 4 套模板（不一定每批都有，根据实际内容提取）：

**模板1：场景证明型**
适合：长焦、夜拍、续航、散热、防抖、防诈
结构：抛真实场景 → 展示痛点 → 产品能力解决 → 给出结果 → 补行动引导

**模板2：人设种草型**
适合：颜值、配色、便携、女性向、生活方式
结构：人设切入 → 生活方式镜头 → 产品成为身份道具 → 顺带证明1个功能 → 落购买动作

**模板3：单卖点成交型**
适合：走量款、价格段产品、直播转化片、渠道型素材
结构：前3秒点题 → 一个核心卖点 → 一个强证据 → 一个使用收益 → 一个促单理由

**模板4：促销引流型**
适合：大促、直播间导流、限时政策
注意：不能单独长期使用，必须叠加产品理由

### Step 5. 输出内容策略配比建议

根据样本分析结果，给出建议配比：

| 类型 | 建议占比 | 说明 |
|------|---------|------|
| 场景证明型 | 40% | 既做欲望，也做说服 |
| 单卖点成交型 | 30% | 转化主力 |
| 人设种草型 | 20% | 品牌高端感 |
| 活动/品牌势能型 | 10% | 节点用 |

### Step 6. 最终一句话结论

用一句核心洞察收尾，说明这批竞品里"真正值得抄的是什么"。

## 种草视频核心公式（底层通用公式）

无论快审还是详审，都可以用此公式做内容质量初判：

**公式 = 场景 × 欲望 × 证据 × 身份**

| 要素 | 含义 | 常见错误 |
|------|------|---------|
| 场景 | 用户真实想进入的生活情境 | 先讲产品，而不是先抛羡慕场景 |
| 欲望 | 在那个场景里"想要"什么感觉 | 欲望没有挂在具体场景上 |
| 证据 | 产品在这个场景里怎么满足欲望 | 只说"很专业"，没有镜头证明 |
| 身份 | 用完之后用户觉得自己是什么样的人 | 没有身份感，只有功能罗列 |

**公式成立的前提**：四个要素必须同时出现，缺任何一个都会变成"空广告"或"纯促销"。

### 实操三步法

1. **先抛"羡慕场景"**，而不是先讲产品
   - ❌ 错误：我们的手机有多强大…
   - ✅ 正确：你是不是也想去演唱会拍清楚爱豆…
2. **让产品成为进入场景的工具**
   - 不是"手机有这个功能"，而是"有了它你可以做到那件事"
3. **用可见证据收口**，而不是用话术收口
   - ❌ 错误：强调"我们的手机很专业"
   - ✅ 正确：拍到鸟的眼睛/拍清演唱会爱豆

### 自家内容检查清单

做一条种草视频前，先问：
1. 有羡慕场景吗？→ 用户看完想说"对，我也想这样"
2. 有产品出现吗？→ 不是品牌广告，是产品在其中扮演角色
3. 有可见证据吗？→ 镜头里真的能看到产品做到了
4. 有用完之后的身份感吗？→ 用户能想象"用完之后我是这种人"
5. 有转化收口吗？→ 知道看完之后可以做什么（预约/下单/进直播间）

五题全YES，才是完整的种草内容。
