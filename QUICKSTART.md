# 快速入门指南

## 5分钟快速开始

### 步骤 1: 安装依赖

打开终端/命令提示符，运行：

```bash
pip install -r requirements.txt
playwright install chromium
```

### 步骤 2: 准备CSV文件

创建或准备包含链接的CSV文件，格式如下：

```csv
链接
https://www.xiaohongshu.com/explore/xxxxx
https://www.xiaohongshu.com/explore/yyyyy
```

### 步骤 3: 启动GUI

```bash
python link_monitor_gui.py
```

### 步骤 4: 配置并运行

1. **选择输入CSV文件** - 点击"浏览"按钮选择你的CSV文件
2. **选择输出目录** - 点击"浏览"按钮选择报告保存位置
3. **点击"立即开始监测"** - 开始处理

### 步骤 5: 查看结果

- 界面会显示实时处理进度
- 完成后会弹出提示框
- 在输出目录中查找生成的Excel报告

## 定时任务使用

### 设置每天9点自动运行

1. 勾选"启用定时运行"
2. 输入时间：`09:00`
3. 点击"立即开始监测"（会启动定时器）

> 注意：程序需要保持运行才能执行定时任务

### 定时任务原理

- 程序会在后台等待设定的时间
- 到达指定时间后自动执行监测
- 完成后继续等待下一天的同一时间
- 每次执行都会生成带时间戳的新报告

## 常用场景

### 场景1: 一次性批量监测

```
1. 准备好CSV文件
2. 启动GUI
3. 选择文件和输出目录
4. 点击"立即开始监测"
5. 等待完成，查看报告
```

### 场景2: 每日定时监测

```
1. 准备好CSV文件
2. 启动GUI
3. 勾选"启用定时运行"
4. 设置运行时间（如09:00）
5. 点击"立即开始监测"
6. 最小化窗口，保持程序运行
7. 每天自动生成报告
```

### 场景3: 需要登录的网站

对于需要登录才能访问的链接：

1. 使用浏览器手动登录网站
2. Playwright会使用相同的浏览器配置
3. 或在代码中添加登录逻辑

## 性能建议

### 少量链接 (< 20)
- 直接运行，预计耗时: 约1-2分钟

### 中量链接 (20-100)
- 建议分批处理
- 预计耗时: 约5-10分钟

### 大量链接 (> 100)
- 强烈建议分批处理
- 可以拆分多个CSV文件
- 预计耗时: 按每个链接5-10秒计算

## 故障排除

### 问题: GUI无法启动
```bash
# 确保安装了tkinter
python -m tkinter  # 应该弹出测试窗口
```

### 问题: 截图失败
```bash
# 重新安装Playwright浏览器
playwright install chromium --force
```

### 问题: 内存占用过高
- 减少同时处理的链接数量
- 分批处理大量链接

### 问题: Excel文件无法打开
- 确保文件没有被其他程序占用
- 检查输出目录的写入权限
- 尝试用最新版本的Excel或WPS打开

## 技巧与建议

### 💡 提示 1: 测试先行
首次使用建议用2-3个链接测试，确认一切正常。

### 💡 提示 2: 定期备份
定期备份生成的Excel报告，避免数据丢失。

### 💡 提示 3: 网络稳定
确保网络连接稳定，避免截图失败。

### 💡 提示 4: 链接有效性
定期检查CSV中的链接是否仍然有效。

### 💡 提示 5: 输出目录管理
定期清理输出目录，避免占用过多磁盘空间。
