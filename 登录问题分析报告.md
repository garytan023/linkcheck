1. **多维度登录状态检测**
```python
def check_login_state(self, page):
    """检查登录状态"""
    login_check = page.evaluate("""() => {
        return {
            hasAvatar: !!document.querySelector('.avatar, .user-avatar'),
            hasUserInfo: !!document.querySelector('.user-info, .login-info'),
            hasLoginBtn: !!document.querySelector('.login-btn, [class*="login-button"]'),
            hasCookie: document.cookie.includes('web_session'),
            isLoggedIn: (avatar || userInfo) && hasCookie && !loginBtn
        };
    }""")
```

2. **完整的登录流程**
```python
def perform_login(self, page):
    """执行完整的登录流程"""

    # 1. 打开小红书首页
    page.goto('https://www.xiaohongshu.com/')

    # 2. 检查历史登录状态
    login_state = self.check_login_state(page)

    # 3. 如未登录，显示扫码界面并等待
    if not login_state.get('isLoggedIn'):
        # 实时倒计时和状态检测
        for remaining in range(wait_seconds, 0, -1):
            if remaining % 5 == 0:
                current_state = self.check_login_state(page)
                if current_state.get('isLoggedIn'):
                    break

    # 4. 保存登录状态
    page.context.storage_state(path=self.storage_state_path)
```
